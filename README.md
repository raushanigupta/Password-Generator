# 🔐 Advanced Password Generator

## 📌 Overview

The Advanced Password Generator is a Python desktop application developed using **Tkinter**. It allows users to generate strong and secure passwords by selecting password length and different character types such as uppercase letters, lowercase letters, digits, and special characters.

The application also provides a password strength indicator and a one-click copy-to-clipboard feature.

---

## ✨ Features

- Generate secure random passwords
- Select password length (4–64 characters)
- Include uppercase letters
- Include lowercase letters
- Include numbers (digits)
- Include special characters
- Password strength indicator (Weak, Moderate, Strong)
- Copy generated password to clipboard
- Simple and user-friendly GUI

## ▶️ How to Run

1. Open the project folder.
2. Open a terminal in the project directory.
3. Run the following command:

```bash
python "Password Generator.py"
```

---

## 📖 How It Works

1. Select the desired password length.
2. Choose the character types to include:
   - Uppercase Letters
   - Lowercase Letters
   - Digits
   - Special Characters
3. Click **Generate Password**.
4. The generated password will appear in the text box.
5. Click **Copy to Clipboard** to copy the password.

---

## 🔒 Password Strength

The application estimates password strength based on password length:

| Password Length | Strength |
|-----------------|----------|
| Less than 8     | Weak     |
| 8–11            | Moderate |
| 12 or more      | Strong   |
